# mdbook
This docs uses [mdbook](https://rust-lang.github.io/mdBook/index.html), to install mdbook you can do either    
install Rust and run `cargo install mdbook`    
or     
on a mac run `brew install mdbook`                                                                                                                                                
# USD8 Docs
`mdbook serve` serve local version.    
`mdbook build` build the book. then just push to github to go live.    
`npm run deploy` runs mdbook build, and git add and commit.
